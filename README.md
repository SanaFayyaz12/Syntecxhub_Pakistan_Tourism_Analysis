# Syntecxhub_Pakistan_Tourism_Analysis
About This Project
This project dives deep into Pakistan's Tourism Data from 2015 to 2024 — analyzing where tourists go, how much they spend, which provinces shine, and what drives popularity scores across the country.Built with Python and presented as a 9-chart interactive dashboard, this analysis reveals insights that matter for tourism policy, travel planning, and business decisions.
